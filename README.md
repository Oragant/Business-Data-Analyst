# Business-Data-Analyst
This project showcases an end-to-end business analytics dashboard built in Microsoft Excel, using transactional order data to deliver clear, executive-level insights.  The dashboard provides a consolidated view of sales performance, customer behaviour, channel effectiveness, and returns, designed to support data-driven decision-making.

# Objectives

Analyse transactional order data to uncover key business trends
Develop an executive-ready Excel dashboard
Apply structured analytics techniques including data modelling, KPI creation, and performance analysis
Demonstrate effective data storytelling using Excel

# Tools & Skills Used

Microsoft Excel
PivotTables and PivotCharts
Data modelling and feature engineering
KPI development
Business performance analysis
Dashboard design and visual storytelling
